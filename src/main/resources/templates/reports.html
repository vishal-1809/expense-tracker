About Us

Welcome to the Expense Tracker — your smart solution for managing personal finances. This web application is designed to help users track their incomes, monitor expenses, and visualize their financial health through interactive charts.

The system provides a comprehensive dashboard where users can view their total income, total expenses, and the number of transactions. All financial data is visually represented using Bar Graphs, Line Graphs, and Pie Charts to ensure clear financial insights.

This project was developed by Vishal Yadav, a Full Stack Java Developer with alost 2 years of experience at Tata Consultancy Services (TCS). It leverages modern technologies such as Java Spring Boot, MySQL, HTML, CSS, and JavaScript to deliver a robust, user-friendly, and performance-optimized personal expense tracking solution.

Technologies Used:
Java Spring Boot (Backend) · MySQL (Database) · HTML, CSS, JavaScript (Frontend) · Chart.js for Graphs

One category → Many transactions
(category_id in transactions refers to id in categories)
categories: Defines classification types for transactions such as "Salary", "Rent", or "Groceries". Each category has a type (either INCOME or EXPENSE).

transactions: Logs each financial record with amount, description, date, and category. Links to categories using category_id foreign key.

The system ensures normalization and integrity by maintaining a clean one-to-many relationship.
